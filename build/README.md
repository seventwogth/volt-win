# Каталог `build`

Каталог `build/` используется `Wails` для сборочных артефактов и платформенных файлов приложения.

## Структура

- `bin/` — результат `wails build`
- `darwin/` — файлы для macOS-сборки
- `windows/` — файлы для Windows-сборки

## Как использовать

- `wails dev` использует платформенные конфигурации из этого каталога
- `wails build` складывает готовые артефакты в `build/bin`
- если платформенные файлы были сильно изменены и нужно вернуться к шаблону Wails по умолчанию, их можно удалить и пересобрать проект

## macOS

Каталог `darwin/` содержит файлы для сборки macOS-приложения:

- `Info.plist` — основной plist для production build
- `Info.dev.plist` — plist для `wails dev`

## Windows

Каталог `windows/` содержит файлы для сборки Windows-версии:

- `icon.ico` — иконка приложения
- `installer/*` — ресурсы для установщика
- `info.json` — метаданные приложения для Windows
- `wails.exe.manifest` — manifest исполняемого файла

Если `icon.ico` отсутствует, `Wails` сможет пересоздать его из `build/appicon.png`.
